# Facebook Login Page Clone (Old UI)

This is a frontend project where I recreated the classic Facebook login page UI using **HTML & CSS**.  
The project helped me improve my frontend skills, especially in **responsive design**, **layout structure**, and **CSS positioning**.

## Features
- Pixel-perfect layout similar to old Facebook login page  
- Fully responsive design (desktop & mobile)  
- Clean and minimal UI  

## Tech Stack
- HTML  
- CSS  


## What I Learned
- CSS Flexbox and positioning  
- Responsive web design principles  
- Understanding classic UI layouts and spacing
